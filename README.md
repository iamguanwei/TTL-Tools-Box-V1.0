# TTL-Tools-Box-V1.0
TTL文字转语音工具包

## 功能介绍
- 支持将文本转换为语音
- 提供多种语音选择
- 支持批量转换
- 支持角色分配

## 未来计划 
- 目前仅支持特别定制的TTL引擎，未来或许会支持通过配置文件连接新的
- 个人的第一个使用AI（80%）完成的作品，简单尝试而已

## 技术栈
- C#
- .NET Framework

## 安装与使用
1.  克隆本仓库到本地
2.  使用 Visual Studio 打开 `TTL Tools Box V1.0.sln`
3.  编译并运行项目
