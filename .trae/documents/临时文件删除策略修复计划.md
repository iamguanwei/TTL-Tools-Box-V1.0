# 临时文件删除策略修复计划

## 一、问题分析

### 1.1 当前问题

大多数情况下删除任务时临时文件没有被正确删除。

### 1.2 根本原因

当 `task.Speed == 0` 时，`AdjustItemsAudioSpeedAsync` 方法会修改 `item.TempFile` 为调整后的文件路径，导致**原始临时文件路径（TTL引擎生成的文件）丢失**，无法被删除。

### 1.3 当前代码问题

1. 没有保存原始临时文件路径
2. 删除策略不符合用户要求

***

## 二、最终策略（用户指定）

1. **任务完成时**，如果"保留临时文件"复选框**没有被选中**，那么将**删除所有临时文件**。
2. **任务完成时**，如果"保留临时文件"复选框**被选中**，那么将**删除除"temp\_"开头的所有临时文件**。
3. **预览任务完成时**，也参照上面策略。
4. **任务删除时**，**删除所有临时文件**（无论之前是否删除，都要尝试删除）。

***

#### 3.2.2 创建统一的临时文件清理方法

合并重复代码，创建统一的清理方法。

```csharp
/// <summary>
/// 清理任务的临时文件。
/// </summary>
/// <param name="task">任务。</param>
/// <param name="tempFilesToCleanup">需要清理的其他临时文件列表。</param>
/// <param name="keepTempFiles">是否保留temp_开头的临时文件。</param>
/// <param name="deleteAll">是否删除所有文件（任务删除时使用）。</param>
private void cleanupTempFiles(VoiceGenerationTask task, List<string> tempFilesToCleanup, bool keepTempFiles, bool deleteAll)
{
    // 收集所有需要删除的文件
    HashSet<string> filesToDelete = new HashSet<string>();

    // 1. 收集任务项的临时文件
    foreach (var item in task.Items)
    {
        // 优先使用原始临时文件路径（因为TempFile可能被修改）
        string originalFile = item.OriginalTempFile;
        string currentFile = item.TempFile;

        if (!string.IsNullOrWhiteSpace(originalFile) && File.Exists(originalFile))
        {
            filesToDelete.Add(originalFile);
        }
        if (!string.IsNullOrWhiteSpace(currentFile) && File.Exists(currentFile))
        {
            filesToDelete.Add(currentFile);
        }
    }

    // 2. 收集其他临时文件
    foreach (string tempFile in tempFilesToCleanup)
    {
        if (!string.IsNullOrWhiteSpace(tempFile) && File.Exists(tempFile))
        {
            filesToDelete.Add(tempFile);
        }
    }

    // 3. 根据策略删除文件
    foreach (string file in filesToDelete)
    {
        string fileName = Path.GetFileName(file);
        bool isTempFile = fileName.StartsWith("temp_", StringComparison.OrdinalIgnoreCase);

        // 删除策略：
        // - deleteAll=true：删除所有文件
        // - keepTempFiles=true且isTempFile=true：保留temp_开头的文件
        // - 其他情况：删除文件
        bool shouldDelete = deleteAll || !(keepTempFiles && isTempFile);

        if (shouldDelete)
        {
            try { File.Delete(file); } catch { }
        }
    }
}
```

#### 3.2.3 修改 finally 块的清理逻辑

使用统一的清理方法替换重复代码。

**当前代码**（第632-678行）：


**修改为**：

```csharp
finally
{
    if (_currentTaskDeleted)
    {
        // 任务删除时：删除所有临时文件
        cleanupTempFiles(task, tempFilesToCleanup, false, true);
        _currentTaskDeleted = false;
    }
    else if (success)
    {
        // 任务完成时：根据"保留临时文件"复选框决定删除策略

        cleanupTempFiles(task, tempFilesToCleanup, keepTempFiles, false);

        {
```
