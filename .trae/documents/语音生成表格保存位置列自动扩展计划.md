# 计划：设置"语音生成"表格"保存位置"列自动扩展

## 需求分析
将"语音生成"表格中的"保存位置"列设置为根据所有单元格宽度自动扩展。

## 当前状态
- 文件位置：`GW.TTLtoolsBox.WinFormUi\UI\Panels\VoiceGenerationPanel.cs`
- 方法位置：`init语音生成Ui()` 方法（第339-355行）
- 当前设置：`Width = 150`（固定宽度150像素）

## 实现方案

### 修改内容
修改"保存位置"列（FileName列）的配置，将其 `AutoSizeMode` 设置为 `DataGridViewAutoSizeColumnMode.AllCells`，使其根据所有单元格（包括标题和所有数据行）的内容宽度自动调整。

### 具体修改
将第349行：
```csharp
dgv_语音生成_任务清单.Columns.Add(new DataGridViewTextBoxColumn { Name = "FileName", DataPropertyName = "FileName", HeaderText = "保存位置", Width = 150, ReadOnly = true });
```

修改为：
```csharp
dgv_语音生成_任务清单.Columns.Add(new DataGridViewTextBoxColumn { Name = "FileName", DataPropertyName = "FileName", HeaderText = "保存位置", AutoSizeMode = DataGridViewAutoSizeColumnMode.AllCells, ReadOnly = true });
```

### 说明
- `DataGridViewAutoSizeColumnMode.AllCells`：列宽自动调整以适应所有单元格的内容，包括标题单元格和所有数据单元格
- 移除固定的 `Width = 150` 设置
- 添加 `AutoSizeMode = DataGridViewAutoSizeColumnMode.AllCells` 属性

## 实施步骤
1. 打开 `VoiceGenerationPanel.cs` 文件
2. 定位到 `init语音生成Ui()` 方法中的 FileName 列定义
3. 修改列配置，设置自动扩展模式
