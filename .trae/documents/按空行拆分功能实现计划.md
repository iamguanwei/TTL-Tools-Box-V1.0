# 按空行拆分功能实现计划

## 需求概述
在"文字拆分"面板上实现"按空行拆分"功能：
- 逻辑：遇到空行（空白行）才拆分成段，并替换掉一段内的所有回车换行符
- 示例："今天天气很好\r\n明天天气不好\r\n\r\n后天天气也很好\r\n" → 拆分为两段：
  - 第一段："今天天气很好明天天气不好\r\n"
  - 第二段："后天天气也很好"

## 实现步骤

### 步骤1：在TextSplitHelper中添加按空行拆分方法
**文件**: `GW.TTLtoolsBox.Core\TextSplit\TextSplitHelper.cs`

添加一个新的静态方法 `SplitByEmptyLine`：
- 参数：
  - `originalText`: 原始文本
  - `trimSegments`: 是否移除首尾空白符（默认false）
- 返回：拆分后的文本（段落间用\r\n分隔）
- 实现逻辑：
  1. 按`\r\n`、`\n`、`\r`拆分文本为行数组
  2. 遍历行数组，将非空行累积到当前段落
  3. 遇到空行时，将当前段落输出（段落内换行符已被移除）
  4. 最后输出剩余段落

## 代码修改详情

### 1. TextSplitHelper.cs 新增方法

```csharp
/// <summary>
/// 按空行拆分文本。
/// </summary>
/// <param name="originalText">原始文本。</param>
/// <param name="trimSegments">是否移除拆分后段落的首尾空白符（默认false）。</param>
/// <returns>拆分后的文本（段落间用\r\n分隔）。</returns>
public static string SplitByEmptyLine(string originalText, bool trimSegments = false)
{
    if (string.IsNullOrEmpty(originalText)) return string.Empty;

    // 统一按换行符拆分为行数组
    string[] lines = originalText.Split(new[] { "\r\n", "\n", "\r" }, StringSplitOptions.None);
    List<string> paragraphs = new List<string>();
    StringBuilder currentParagraph = new StringBuilder();

    foreach (string line in lines)
    {
        if (string.IsNullOrWhiteSpace(line))
        {
            // 遇到空行，输出当前段落（如果有内容）
            if (currentParagraph.Length > 0)
            {
                string paragraph = trimSegments ? currentParagraph.ToString().Trim() : currentParagraph.ToString();
                paragraphs.Add(paragraph);
                currentParagraph.Clear();
            }
        }
        else
        {
            // 非空行，追加到当前段落（移除换行符效果）
            currentParagraph.Append(line);
        }
    }

    // 输出最后一个段落（如果有内容）
    if (currentParagraph.Length > 0)
    {
        string lastParagraph = trimSegments ? currentParagraph.ToString().Trim() : currentParagraph.ToString();
        paragraphs.Add(lastParagraph);
    }

    return string.Join("\r\n", paragraphs);
}
```
