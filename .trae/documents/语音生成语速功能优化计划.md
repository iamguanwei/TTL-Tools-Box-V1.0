# 语音生成语速功能优化计划

## 一、需求分析

### 1.1 当前问题

1. 默认朗读者下拉列表显示格式不包含语速信息
2. 缺少"指定全局语速"功能，无法统一覆盖所有朗读者的语速
3. 任务项没有独立的 Speed 属性，提交后朗读者的语速可能被修改
4. 语速处理逻辑需要调整，支持按任务项单独调整语速

### 1.2 目标功能

1. 默认朗读者下拉列表格式改为"朗读者源名称 \[语速100%]"
2. 添加"指定全局语速"复选框，控制是否使用全局语速覆盖
3. 为任务项添加独立 Speed 属性，保存提交时的语速值
4. 发送时根据全局语速设置决定发送值（0表示不使用，>0表示实际语速）
5. 调整任务执行顺序：Speed=0时先调整每条语音语速再合并
6. 任务列表显示：Speed>0显示具体值，Speed=0显示"-"

### 2.6 VoiceGenerationTaskQueue.cs（任务队列管理器）

#### 2.6.1 修改音频处理逻辑

* **位置**：`ExecuteTaskAsync()` 方法（第540-555行附近）

* **当前代码**：

```csharp
task.ProgressDetail = "正在合并音频...";
OnTaskProgressUpdated(task, task.Progress, task.ProgressDetail);

```

* **修改为**：

```csharp
if (task.Speed > 0)
{
    // 使用全局语速：先合并后统一调整语速
    task.ProgressDetail = "正在合并音频...";
    OnTaskProgressUpdated(task, task.Progress, task.ProgressDetail);

    mergedFile = await MergeAudioFilesAsync(task.Items, task.SpaceTime, task.SaveFile);
    tempFilesToCleanup.Add(mergedFile);

    if (_shouldStop) return;

    task.ProgressDetail = "正在调整音频速度...";
    OnTaskProgressUpdated(task, task.Progress, task.ProgressDetail);

    speedAdjustedFile = await AdjustAudioSpeedAsync(mergedFile, task.Speed);
    if (speedAdjustedFile != mergedFile)
    {
        tempFilesToCleanup.Add(speedAdjustedFile);
    }
}
else
{
    // 使用各自朗读者的语速：先调整每条语音语速，再合并
    task.ProgressDetail = "正在调整各段音频速度...";
    OnTaskProgressUpdated(task, task.Progress, task.ProgressDetail);

    // 调整每个任务项的语速
    var adjustedItems = await AdjustItemsAudioSpeedAsync(task.Items);
    foreach (var adjustedFile in adjustedItems.Where(f => f != null))
    {
        tempFilesToCleanup.Add(adjustedFile);
    }

    if (_shouldStop) return;

    task.ProgressDetail = "正在合并音频...";
    OnTaskProgressUpdated(task, task.Progress, task.ProgressDetail);

    mergedFile = await MergeAudioFilesAsync(task.Items, task.SpaceTime, task.SaveFile);
    tempFilesToCleanup.Add(mergedFile);

    speedAdjustedFile = mergedFile;  // 不需要再统一调整语速
}
```

#### 2.6.2 添加单独调整语速的方法

* **位置**：新增私有方法

* **新增代码**：

```csharp
/// <summary>
/// 调整多个任务项音频的语速。
/// </summary>
/// <param name="taskItems">任务项数组。</param>
/// <returns>调整后的临时文件路径列表。</returns>
private async Task<string[]> AdjustItemsAudioSpeedAsync(VoiceGenerationTaskItem[] taskItems)
{
    return await Task.Run(() =>
    {
        List<string> adjustedFiles = new List<string>();
        
        for (int i = 0; i < taskItems.Length; i++)
        {
            VoiceGenerationTaskItem item = taskItems[i];
            
            if (string.IsNullOrWhiteSpace(item.TempFile) || !File.Exists(item.TempFile))
            {
                adjustedFiles.Add(null);
                continue;
            }

            int itemSpeed = item.Speed > 0 ? item.Speed : 100;
            
            if (itemSpeed == 100)
            {
                // 语速为100，不需要调整
                adjustedFiles.Add(null);
                continue;
            }

            // 生成调整后的文件路径（使用 speed_item_ 前缀避免冲突）
            string adjustedFile = Path.Combine(_tempFolder, $"speed_item_{Guid.NewGuid()}.wav");
            float tempo = itemSpeed / 100f;
            string inputFileName = Path.GetFileName(item.TempFile);
            string outputFileName = Path.GetFileName(adjustedFile);
            string ffmpegArgs = $"-i \"{inputFileName}\" -filter:a \"atempo={tempo:F2}\" -y \"{outputFileName}\"";

            RunFFmpegCommand(ffmpegArgs, _tempFolder).Wait();

            // 更新任务项的临时文件路径
            item.TempFile = adjustedFile;
            adjustedFiles.Add(adjustedFile);
        }

        return adjustedFiles.ToArray();
    });
}
```

## 三、潜在问题与解决方案

### 3.1 SpeakerInfo 显示格式问题

* **问题**：SpeakerInfo 类的 ToString() 方法已被用于序列化，不能直接修改

* **方案**：使用 ComboBox 的 Format 事件自定义显示格式

### 3.2 语速调整文件命名冲突

* **问题**：调整语速后的临时文件可能与 TTL 引擎生成的临时文件重名

* **方案**：使用不同的前缀（`speed_item_` 而非 `temp_`）

### 3.3 向后兼容性

* **问题**：旧版本保存的任务数据没有 Speed 属性

* **方案**：Speed 属性默认值为 0，旧数据加载时使用默认值

### 3.4 任务项语速来源

* **问题**：任务项的 Speed 应该从哪里获取

* **方案**：

  * 任务项的 Speed 始终从 Speaker.Speed 获取（提交时保存）

  * 任务级别的 Speed 表示是否使用全局语速（>0全局，0各自）

### 3.5 发现的额外问题

* **问题**：当前 `fill语音生成任务清单` 方法中 speed 参数用于任务级别，需要区分

* **方案**：

  * speed > 0：表示使用全局语速，任务.Speed = speed，任务项.Speed = Speaker.Speed

  * speed == 0：表示不使用全局语速，任务.Speed = 0，任务项.Speed = Speaker.Speed
