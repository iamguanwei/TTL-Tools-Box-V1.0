# 语速显示和预览文件修改计划

## 一、需求分析

### 1.1 修改内容
1. "语音生成预处理"面板的"默认朗读者"下拉列表框的项的格式，改回最初的"朗读者源名称"
2. 朗读者声音预览文件加上速度，在原来的文件名后面加上"_速度"，不同速度对应不同的预览文件

### 1.2 当前状态
- VoicePreprocessPanel.cs 中添加了 Format 事件，显示格式为"朗读者源名称 [语速%]"
- PreviewVoiceManager.cs 中预览文件命名格式为 `{engineId}_{safeSourceName}.mp3`

---

## 二、修改方案

### 2.1 VoicePreprocessPanel.cs（语音生成预处理面板）

#### 2.1.1 移除 Format 事件处理
- **位置**：`init语音生成预处理Ui()` 方法
- **当前代码**（第240行）：
```csharp
this.cb_语音生成预处理_默认朗读者设置.Format += cb_语音生成预处理_默认朗读者设置_Format;
```
- **修改方案**：删除此行

#### 2.1.2 删除 Format 事件处理方法
- **位置**：事件处理区域
- **删除方法**：
```csharp
private void cb_语音生成预处理_默认朗读者设置_Format(object sender, ListControlConvertEventArgs e)
{
    if (e.ListItem is SpeakerInfo speaker)
    {
        e.Value = $"{speaker.SourceName} [{speaker.Speed}%]";
    }
}
```
