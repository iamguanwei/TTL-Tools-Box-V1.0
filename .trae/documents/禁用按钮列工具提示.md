# 禁用按钮列工具提示的实现计划

## 问题分析

DataGridView 有一个 `ShowCellToolTips` 属性，默认为 `true`。当此属性为 `true` 时，DataGridView 会自动为单元格显示工具提示：
- 对于按钮列，按钮上的文字就是单元格的值
- DataGridView 会自动将这个值作为工具提示显示
- 所以工具提示内容和按钮文字完全一样

之前尝试的 `CellFormatting` 和 `CellToolTipTextNeeded` 事件方法可能被这个自动行为覆盖。

## 解决方案

将 DataGridView 的 `ShowCellToolTips` 属性设置为 `false`，禁用所有单元格的自动工具提示功能。

### 实现步骤

#### 1. TtlSchemePanel.cs 修改

在 `setupSpeakerGridColumns()` 方法中，添加以下设置：

```csharp
this.dgv_TTL方案_朗读者参数配置.ShowCellToolTips = false;
```

#### 2. RoleMappingPanel.cs 修改

在 `setup角色映射GridColumns()` 方法中，添加以下设置：

```csharp
this.dgv_角色映射.ShowCellToolTips = false;
```

## 涉及文件

1. `GW.TTLtoolsBox.WinFormUi\UI\Panels\TtlSchemePanel.cs`
2. `GW.TTLtoolsBox.WinFormUi\UI\Panels\RoleMappingPanel.cs`

## 注意事项

- `ShowCellToolTips = false` 会禁用整个表格所有单元格的工具提示
- 如果其他列需要保留工具提示，则需要使用 `CellToolTipTextNeeded` 事件单独处理
