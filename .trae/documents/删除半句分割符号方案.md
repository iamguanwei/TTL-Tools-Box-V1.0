# 删除"Constants.半句_分割符号"及相关逻辑方案

## 一、背景分析

### 1.1 当前代码结构

**Constants.cs** (第21行)
```csharp
public const string 半句_分割符号 = "，";
```

**TextSplitPanel.cs** 中使用位置：
- 第42行：定义本地常量 `private const string 半句_分割符号 = Constants.半句_分割符号;`
- 第192行：UI文本显示 `this.rb_拆分方式_按句子拆分.Text += $" [ 整句切分 {整句_分割符号}  半句切分 {半句_分割符号} ]";`
- 第315行：调用 `TextSplitHelper.SplitTextIgnoreLineBreaks(..., 半句_分割符号.ToCharArray())`
- 第324行：调用 `TextSplitHelper.SplitText(..., 半句_分割符号.ToCharArray())`

**TextSplitHelper.cs** 中的拆分逻辑：
- `SplitText` 方法接收 `mainSeparators`（主分隔符/整句）和 `subSeparators`（次分隔符/半句）
- 拆分策略：先尝试用主分隔符拆分，找不到时再用次分隔符作为"兜底"

### 1.2 影响范围

删除半句分割符号后，文本拆分将只使用整句分割符号（`！!？?。：""…;;`），不再使用逗号作为兜底拆分点。

private static int findFirstSeparatorAfterPos(string text, int startPos, HashSet<char> mainSep)
{
    for (int i = startPos; i < text.Length; i++)
    {
        if (mainSep.Contains(text[i])) return i;
    }
    return -1;
}
