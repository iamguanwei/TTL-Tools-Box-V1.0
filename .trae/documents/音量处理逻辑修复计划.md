# 音量处理逻辑分析与修复计划

## 问题分析

### 当前 Speed 和 Volume 的处理逻辑

| 条件 | Speed 处理 | Volume 处理 |
|------|-------------|--------------|
| `task.Speed > 0` 且 `task.Volume > 0` | 合并 → 统一调整速度 → 统一调整音量 | ✅ 正确 |
| `task.Speed > 0` 且 `task.Volume == 0` | 合并 → 统一调整速度 → **不调整音量** | ❌ 问题！ |
| `task.Speed == 0` 且 `task.Volume > 0` | 各段调整速度 → 各段调整音量 → 合并 | ⚠️ 全局音量被忽略 |
| `task.Speed == 0` 且 `task.Volume == 0` | 各段调整速度 → 各段调整音量 → 合并 | ✅ 正确 |

### 问题根源

**场景：设置了全局速度但没有设置全局音量**

1. 当前逻辑：
   - 先合并音频（不处理各段的音量）
   - 统一调整速度
   - 因为 `task.Volume == 0`，跳过音量调整
   - **结果**：各段的 `item.Volume` 完全被忽略

2. 用户期望：
   - 没有全局音量时，应该使用各段自己的音量
   - 但因为已经合并成一个文件，无法再单独调整各段音量

### 正确的逻辑应该是

| 条件 | 处理流程 |
|------|----------|
| `task.Speed > 0` 且 `task.Volume > 0` | 合并 → 统一调整速度 → 统一调整音量 |
| `task.Speed > 0` 且 `task.Volume == 0` | **先调整各段音量** → 合并 → 统一调整速度 |
| `task.Speed == 0` 且 `task.Volume > 0` | 各段调整速度 → **统一调整音量（合并后）** → 合并 |
| `task.Speed == 0` 且 `task.Volume == 0` | 各段调整速度 → 各段调整音量 → 合并 |

**关键原则**：
- **有全局音量**：合并后统一调整
- **无全局音量**：合并前单独调整各段

## 修复方案

### 文件： VoiceGenerationTaskQueue.cs

### 修改位置: ProcessNextTaskAsync 方法中的音频处理逻辑

### 修改内容

```csharp
string volumeAdjustedFile = null;

if (task.Speed > 0)
{
    // 有全局速度设置
    if (task.Volume > 0)
    {
        // 有全局速度和全局音量：合并 -> 调整速度 -> 调整音量
        task.ProgressDetail = "正在合并音频...";
        OnTaskProgressUpdated(task, task.Progress, task.ProgressDetail);
        mergedFile = await MergeAudioFilesAsync(task.Items, task.SpaceTime, task.SaveFile);
        tempFilesToCleanup.Add(mergedFile);
        if (_shouldStop) return;

        task.ProgressDetail = "正在调整音频速度...";
        OnTaskProgressUpdated(task, task.Progress, task.ProgressDetail);
        speedAdjustedFile = await AdjustAudioSpeedAsync(mergedFile, task.Speed);
        if (speedAdjustedFile != mergedFile)
        {
            tempFilesToCleanup.Add(speedAdjustedFile);
        }
        if (_shouldStop) return;

        task.ProgressDetail = "正在调整音频音量...";
        OnTaskProgressUpdated(task, task.Progress, task.ProgressDetail);
        volumeAdjustedFile = await AdjustAudioVolumeAsync(speedAdjustedFile, task.Volume);
        if (volumeAdjustedFile != speedAdjustedFile)
        {
            tempFilesToCleanup.Add(volumeAdjustedFile);
        }
    }
    else
    {
        // 有全局速度但没有全局音量：先调整各段音量 -> 合并 -> 调整速度
        task.ProgressDetail = "正在调整各段音频音量...";
        OnTaskProgressUpdated(task, task.Progress, task.ProgressDetail);
        var adjustedVolumeFiles = await AdjustItemsAudioVolumeAsync(task.Items);
        foreach (var adjustedFile in adjustedVolumeFiles.Where(f => f != null))
        {
            tempFilesToCleanup.Add(adjustedFile);
        }
        if (_shouldStop) return;

        task.ProgressDetail = "正在合并音频...";
        OnTaskProgressUpdated(task, task.Progress, task.ProgressDetail);
        mergedFile = await MergeAudioFilesAsync(task.Items, task.SpaceTime, task.SaveFile);
        tempFilesToCleanup.Add(mergedFile);
        if (_shouldStop) return;

        task.ProgressDetail = "正在调整音频速度...";
        OnTaskProgressUpdated(task, task.Progress, task.ProgressDetail);
        volumeAdjustedFile = await AdjustAudioSpeedAsync(mergedFile, task.Speed);
        if (volumeAdjustedFile != mergedFile)
        {
            tempFilesToCleanup.Add(volumeAdjustedFile);
        }
    }
}
else
{
    // 没有全局速度设置
    task.ProgressDetail = "正在调整各段音频速度...";
    OnTaskProgressUpdated(task, task.Progress, task.ProgressDetail);
    var adjustedSpeedFiles = await AdjustItemsAudioSpeedAsync(task.Items);
    foreach (var adjustedFile in adjustedSpeedFiles.Where(f => f != null))
    {
        tempFilesToCleanup.Add(adjustedFile);
    }
    if (_shouldStop) return;

    if (task.Volume > 0)
    {
        // 有全局音量：合并后统一调整
        task.ProgressDetail = "正在合并音频...";
        OnTaskProgressUpdated(task, task.Progress, task.ProgressDetail);
        mergedFile = await MergeAudioFilesAsync(task.Items, task.SpaceTime, task.SaveFile);
        tempFilesToCleanup.Add(mergedFile);
        if (_shouldStop) return;

        task.ProgressDetail = "正在调整音频音量...";
        OnTaskProgressUpdated(task, task.Progress, task.ProgressDetail);
        volumeAdjustedFile = await AdjustAudioVolumeAsync(mergedFile, task.Volume);
        if (volumeAdjustedFile != mergedFile)
        {
            tempFilesToCleanup.Add(volumeAdjustedFile);
        }
    }
    else
    {
        // 没有全局音量：先调整各段音量 -> 合并
        task.ProgressDetail = "正在调整各段音频音量...";
        OnTaskProgressUpdated(task, task.Progress, task.ProgressDetail);
        var adjustedVolumeFiles = await AdjustItemsAudioVolumeAsync(task.Items);
        foreach (var adjustedFile in adjustedVolumeFiles.Where(f => f != null))
        {
            tempFilesToCleanup.Add(adjustedFile);
        }
        if (_shouldStop) return;

        task.ProgressDetail = "正在合并音频...";
        OnTaskProgressUpdated(task, task.Progress, task.ProgressDetail);
        mergedFile = await MergeAudioFilesAsync(task.Items, task.SpaceTime, task.SaveFile);
        tempFilesToCleanup.Add(mergedFile);
        volumeAdjustedFile = mergedFile;
    }
}
```

## 验证要点

1. **场景1**：全局速度=100，无全局音量，朗读者音量=50
   - 应该：先调整各段音量到50% → 合并 → 调整速度到100%
   - 结果：音量应该是50%

2. **场景2**：全局速度=100，全局音量=100，朗读者音量=50
   - 应该：合并 → 调整速度到100% → 调整音量到100%
   - 结果：音量应该是100%（全局覆盖）

3. **场景3**：无全局速度，全局音量=100，朗读者音量=50
   - 应该：调整各段速度 → 合并 → 调整音量到100%
   - 结果：音量应该是100%（全局覆盖）

4. **场景4**：无全局速度，无全局音量，朗读者音量=50
   - 应该：调整各段速度 → 调整各段音量到50% → 合并
   - 结果：音量应该是50%
