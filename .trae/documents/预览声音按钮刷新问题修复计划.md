# 预览声音按钮刷新问题修复计划

## 问题分析

### 问题1：重新加载朗读者按钮不能刷新预览声音的播放按钮

**现象**：删除预览文件后，点击"重新加载朗读者"按钮，按钮依然显示"播放"而不是"生成"。

**根本原因**：

1. `bt_TTL方案_重新加载朗读者_Click` 调用 `createSpeakersFromRoles(currentEngine, true)`
2. `createSpeakersFromRoles` 调用 `bindSpeakersToGrid(ttl)`，后者调用 `update自动生成全部朗读者预览音频按钮状态(true)`
3. **但是** `update自动生成全部朗读者预览音频按钮状态` 使用的是 `currentEngine.Speakers`（引擎原始数据）
4. 而 `getPreviewVoiceStatus` 方法中通过 `_findSpeakerByShowName?.Invoke(sourceName)` 获取 SpeakerInfo
5. 这个委托返回的是引擎中的原始 SpeakerInfo，而不是表格中已编辑的数据
6. 表格绑定的是克隆的数据，与引擎原始数据是分离的

**关键代码位置**：

* [TtlSchemePanel.cs:1107-1119](file:///d:/User%20Data/guanw@hotmail.com/我的私有云/GW's%20Data/我的开发/GWs/TTL工具箱/TTL%20Tools%20Box%20V1.0/GW.TTLtoolsBox.WinFormUi/UI/Panels/TtlSchemePanel.cs#L1107-L1119) - `getPreviewVoiceStatus` 方法

### 问题2：更改速度/音量后按钮没有刷新

**现象**：在生成预览音频时，更改速度后按钮没有变成"生成"，但占位文件却自动改成了新的名称。

**根本原因**：

1. 当前只有 `CellValidating` 事件处理，仅用于输入验证
2. 没有 `CellEndEdit` 事件来触发按钮状态刷新
3. `getPreviewVoiceStatus` 方法从 `_findSpeakerByShowName` 获取 SpeakerInfo，返回的是引擎原始数据
4. 表格中编辑后的 Speed/Volume 值没有被正确使用

**关键代码位置**：

* [TtlSchemePanel.cs:1449-1488](file:///d:/User%20Data/guanw@hotmail.com/我的私有云/GW's%20Data/我的开发/GWs/TTL工具箱/TTL%20Tools%20Box%20V1.0/GW.TTLtoolsBox.WinFormUi/UI/Panels/TtlSchemePanel.cs#L1449-L1488) - `CellValidating` 事件

## 修复方案

### 修改1：修复 `getPreviewVoiceStatus` 方法

**文件**：`TtlSchemePanel.cs`

**修改内容**：直接从表格行获取 SpeakerInfo，而不是通过委托从引擎获取

```csharp
private PreviewVoiceStatus getPreviewVoiceStatus(string sourceName)
{
    string engineId = getCurrentEngineId();
    if (string.IsNullOrWhiteSpace(engineId) || string.IsNullOrWhiteSpace(sourceName))
    {
        return PreviewVoiceStatus.未生成;
    }

    // 直接从表格行获取 SpeakerInfo，确保使用的是当前显示的数据
    SpeakerInfo speakerInfo = null;
    foreach (DataGridViewRow row in this.dgv_TTL方案_朗读者参数配置.Rows)
    {
        if (row.DataBoundItem is SpeakerInfo speaker && speaker.SourceName == sourceName)
        {
            speakerInfo = speaker;
            break;
        }
    }

    int speed = speakerInfo?.Speed ?? 100;
    int volume = speakerInfo?.Volume ?? 100;

    return PreviewVoiceManager?.GetStatus(engineId, sourceName, speed, volume) ?? PreviewVoiceStatus.未生成;
}
```

### 修改2：添加 `CellEndEdit` 事件处理

**文件**：`TtlSchemePanel.cs`

**修改内容**：在单元格编辑结束时刷新对应行的按钮状态

1. 添加事件处理方法：

```csharp
/// <summary>
/// 事件处理：单元格编辑结束，用于刷新预览按钮状态。
/// </summary>
private void dgv_TTL方案_朗读者参数配置_CellEndEdit(object sender, DataGridViewCellEventArgs e)
{
    var speedColumn = this.dgv_TTL方案_朗读者参数配置.Columns["Speed"];
    var volumeColumn = this.dgv_TTL方案_朗读者参数配置.Columns["Volume"];
    
    if ((speedColumn != null && e.ColumnIndex == speedColumn.Index) ||
        (volumeColumn != null && e.ColumnIndex == volumeColumn.Index))
    {
        if (e.RowIndex >= 0)
        {
            // 刷新当前行以更新预览按钮状态
            this.dgv_TTL方案_朗读者参数配置.InvalidateRow(e.RowIndex);
            
            // 更新"自动生成全部朗读者预览音频"按钮状态
            update自动生成全部朗读者预览音频按钮状态();
        }
    }
}
```

1. 在 `setupSpeakerGridColumns` 方法中绑定事件：

```csharp
this.dgv_TTL方案_朗读者参数配置.CellEndEdit += dgv_TTL方案_朗读者参数配置_CellEndEdit;
```

### 修改3：修复 `update自动生成全部朗读者预览音频按钮状态` 方法

**文件**：`TtlSchemePanel.cs`

**修改内容**：从表格数据源获取 SpeakerInfo，而不是从引擎获取

```csharp
private void update自动生成全部朗读者预览音频按钮状态(bool refreshStatus = false)
{
    bool hasSpeakersWithoutPreview = false;
    string engineId = getCurrentEngineId();
    
    // 从表格数据源获取 SpeakerInfo
    foreach (DataGridViewRow row in this.dgv_TTL方案_朗读者参数配置.Rows)
    {
        if (row.DataBoundItem is SpeakerInfo speaker)
        {
            PreviewVoiceStatus status;
            if (refreshStatus)
            {
                status = PreviewVoiceManager?.RefreshStatus(engineId, speaker.SourceName, speaker.Speed, speaker.Volume) ?? PreviewVoiceStatus.未生成;
            }
            else
            {
                status = PreviewVoiceManager?.GetStatus(engineId, speaker.SourceName, speaker.Speed, speaker.Volume) ?? PreviewVoiceStatus.未生成;
            }
            if (status == PreviewVoiceStatus.未生成)
            {
                hasSpeakersWithoutPreview = true;
            }
        }
    }
    
    this.bt_TTL方案_自动生成全部朗读者预览音频.Enabled = hasSpeakersWithoutPreview;
}
```

1. **问题1验证**：

   * 为某个朗读者生成预览音频（速度100，音量100）

   * 修改速度为150，点击其他单元格完成编辑

   * 确认预览按钮从"播放"变为"生成"

   * 点击"播放"应能播放之前生成的音频（速度100）

   * 点击"生成"应生成新的音频（速度150）
