# 语音生成文件名保存策略优化计划（修订版）

## 一、需求理解

### 核心规则

1. **单个任务**：直接使用原始文件名，不做任何处理

2. **多个任务**：
   - **文件名末尾有数字**：从该数字开始累加
   - **文件名末尾无数字**：第一个任务不加数字，后续任务从1开始累加

3. **数字位数规则**：
   - 数字位数 = max(用户输入数字位数, 任务数量所需位数)
   - 用户输入 `1`（1位），3个任务 → 输出 `1`, `2`, `3`（1位，因为3个任务只需1位）
   - 用户输入 `01`（2位），3个任务 → 输出 `01`, `02`, `03`（2位，保留用户指定的位数）
   - 用户输入 `1`（1位），30个任务 → 输出 `01`, `02`...`30`（2位，因为30个任务需要2位）

```csharp
// 计算数字位数
int taskCountDigits = paragraphArray.Length.ToString().Length;

if (isSingleTask)
{
    // 单任务：直接使用原始文件名
    task.SaveFile = Path.Combine(folderPath, $"{mFileName}{eFileName}");
}
else
{
    // 多任务：解析末尾数字
    int fileCount = 1;
    string baseFileName = mFileName;
    int userDigits = 0;  // 用户输入的数字位数
    bool hasTrailingNumber = false;

    var strArray = FileHelper.SplitTrailingNumber(mFileName);
    if (strArray != null && strArray.Length > 1)
    {
        baseFileName = strArray[0];
        fileCount = int.Parse(strArray[1]);
        userDigits = strArray[1].Length;  // 用户输入的数字位数
        hasTrailingNumber = true;
    }

    // 数字位数 = max(用户输入位数, 任务数量所需位数)
    int digitCount = Math.Max(userDigits, taskCountDigits);

    int taskIndex = 0;
    foreach (var parapraph in paragraphArray)
    {
        string fileNameSuffix;
        if (taskIndex == 0 && !hasTrailingNumber)
        {
            // 第一个任务且用户未提供数字：不加数字
            fileNameSuffix = string.Empty;
        }
        else
        {
            // 其他情况：添加数字
            fileNameSuffix = FileHelper.PadNumber(fileCount++, digitCount);
        }

        task.SaveFile = Path.Combine(folderPath, $"{baseFileName}{fileNameSuffix}{eFileName}");
        taskIndex++;
    }
}
```
