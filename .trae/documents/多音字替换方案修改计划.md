# 多音字替换方案修改计划

## 一、需求分析

### 1.1 用户需求

修改"音调拼音标注"和"数字声调拼音标注"两种多音字替换方案：
- **当前行为**：如果原始文本中所有字都不是多音字，返回原始文本本身
- **期望行为**：如果原始文本中所有字都不是多音字，返回所有原始文本的拼音

### 1.2 当前实现分析

#### 判断是否为多音字的逻辑（两种方案相同）

```csharp
foreach (char ch in originalText)
{
    string[] pinyinArray = PinyinHelper.GetAllPinyinWithToneMark(ch);  // 或 GetAllPinyinWithToneNumber
    if (pinyinArray == null || pinyinArray.Length == 0)
    {
        // 非中文字符：直接返回原字符
        charPinyinList.Add(new List<string> { ch.ToString() });
    }
    else
    {
        var distinctPinyins = pinyinArray.Distinct().ToList();
        if (distinctPinyins.Count == 1)
        {
            // 单音字：直接返回原字符（不是多音字）
            charPinyinList.Add(new List<string> { ch.ToString() });
        }
        else
        {
            // 多音字：保留所有读音
            charPinyinList.Add(distinctPinyins);
        }
    }
}
```

#### 当前问题

当所有字符都不是多音字时，`charPinyinList` 中每个元素的列表都只包含原字符本身，最终生成的组合就是原始文本。用户希望在这种情况下，返回所有字符的拼音。

## 二、修改方案

### 2.1 修改 `TonePinyinPolyphonicScheme.cs`

**修改 `BuildDefaultReplaceStrings` 方法**：

1. 在遍历字符时，记录是否存在多音字
2. 如果不存在多音字，生成一个包含所有字符拼音的替换字符串

**修改后的代码**：

```csharp
public override void BuildDefaultReplaceStrings(string originalText)
{
    if (string.IsNullOrWhiteSpace(originalText) == false)
    {
        List<string> replaceStrList = base.ReplaceStrings.ToList();

        List<List<string>> charPinyinList = new List<List<string>>();
        bool hasPolyphonicChar = false;

        foreach (char ch in originalText)
        {
            string[] pinyinArray = PinyinHelper.GetAllPinyinWithToneMark(ch);
            if (pinyinArray == null || pinyinArray.Length == 0)
            {
                charPinyinList.Add(new List<string> { ch.ToString() });
            }
            else
            {
                var distinctPinyins = pinyinArray.Distinct().ToList();
                if (distinctPinyins.Count == 1)
                {
                    charPinyinList.Add(new List<string> { ch.ToString() });
                }
                else
                {
                    hasPolyphonicChar = true;
                    charPinyinList.Add(distinctPinyins);
                }
            }
        }

        if (hasPolyphonicChar)
        {
            List<string> allCombinations = generatePinyinCombinations(charPinyinList, originalText);
            replaceStrList.AddRange(allCombinations);
        }
        else
        {
            string pinyinCombination = generateAllPinyinCombination(originalText);
            replaceStrList.Add(pinyinCombination);
        }

        base.ReplaceStrings = replaceStrList.Distinct().ToArray();
    }
}

/// <summary>
/// 生成所有字符的拼音组合（用于无非多音字的情况）。
/// </summary>
/// <param name="originalText">原始文本。</param>
/// <returns>拼音组合字符串。</returns>
private string generateAllPinyinCombination(string originalText)
{
    StringBuilder sb = new StringBuilder();
    foreach (char ch in originalText)
    {
        string[] pinyinArray = PinyinHelper.GetAllPinyinWithToneMark(ch);
        if (pinyinArray == null || pinyinArray.Length == 0)
        {
            sb.Append(ch);
        }
        else
        {
            var py = PinyinHelper.SplitPinyin(pinyinArray[0]);
            sb.Append($" [{py.Shengmu}][{py.Yunmu}] ");
        }
    }
    return sb.ToString();
}
```

### 2.2 修改 `NumberTonePinyinPolyphonicScheme.cs`

**修改 `BuildDefaultReplaceStrings` 方法**：

```csharp
public override void BuildDefaultReplaceStrings(string originalText)
{
    if (string.IsNullOrWhiteSpace(originalText) == false)
    {
        List<string> replaceStrList = base.ReplaceStrings.ToList();

        List<List<string>> charPinyinList = new List<List<string>>();
        bool hasPolyphonicChar = false;

        foreach (char ch in originalText)
        {
            string[] pinyinArray = PinyinHelper.GetAllPinyinWithToneNumber(ch);
            if (pinyinArray == null || pinyinArray.Length == 0)
            {
                charPinyinList.Add(new List<string> { ch.ToString() });
            }
            else
            {
                var distinctPinyins = pinyinArray.Distinct().ToList();
                if (distinctPinyins.Count == 1)
                {
                    charPinyinList.Add(new List<string> { ch.ToString() });
                }
                else
                {
                    hasPolyphonicChar = true;
                    charPinyinList.Add(distinctPinyins);
                }
            }
        }

        if (hasPolyphonicChar)
        {
            List<string> allCombinations = generatePinyinCombinations(charPinyinList, originalText);
            replaceStrList.AddRange(allCombinations);
        }
        else
        {
            string pinyinCombination = generateAllPinyinCombination(originalText);
            replaceStrList.Add(pinyinCombination);
        }

        base.ReplaceStrings = replaceStrList.Distinct().ToArray();
    }
}

/// <summary>
/// 生成所有字符的拼音组合（用于无非多音字的情况）。
/// </summary>
/// <param name="originalText">原始文本。</param>
/// <returns>拼音组合字符串。</returns>
private string generateAllPinyinCombination(string originalText)
{
    List<string> pinyins = new List<string>();
    StringBuilder sb = new StringBuilder();

    foreach (char ch in originalText)
    {
        string[] pinyinArray = PinyinHelper.GetAllPinyinWithToneNumber(ch);
        if (pinyinArray == null || pinyinArray.Length == 0)
        {
            if (pinyins.Count > 0)
            {
                sb.Append($"[{string.Join(" ", pinyins)}]");
                pinyins.Clear();
            }
            sb.Append(ch);
        }
        else
        {
            pinyins.Add(pinyinArray[0]);
        }
    }

    if (pinyins.Count > 0)
    {
        sb.Append($"[{string.Join(" ", pinyins)}]");
    }

    return sb.ToString();
}
```

## 四、测试场景

修改完成后，需要测试以下场景：

1. **纯多音字文本**：如"重重"，验证生成多个拼音组合
2. **纯单音字文本**：如"你好"，验证生成一个拼音组合（所有字符的拼音）
3. **混合文本**：如"重好"，包含多音字和单音字，验证按原逻辑处理
4. **包含非中文字符**：如"重a好"，验证非中文字符正确处理
5. **音调拼音标注格式**：验证输出格式为 ` [声母][韵母] `
6. **数字声调拼音标注格式**：验证输出格式为 `[拼音1 拼音2]`
