# 预览任务临时文件删除问题修复计划

## 一、问题分析

### 1.1 当前问题
预览任务删除时，临时文件没有被正确删除。

### 1.2 代码分析

#### 临时文件删除逻辑（第632-645行）
```csharp
if (_currentTaskDeleted)
{
    foreach (var item in task.Items)
    {
        if (!string.IsNullOrWhiteSpace(item.TempFile) && File.Exists(item.TempFile))
        {
            try { File.Delete(item.TempFile); } catch { }
        }
    }
    foreach (string tempFile in tempFilesToCleanup)
    {
        try { if (File.Exists(tempFile)) File.Delete(tempFile); } catch { }
    }
    _currentTaskDeleted = false;
}
```

#### 音频处理逻辑（第559-578行）
当 `task.Speed == 0` 时（使用各自朗读者的语速）：
```csharp
var adjustedFiles = await AdjustItemsAudioSpeedAsync(task.Items);
foreach (var adjustedFile in adjustedFiles.Where(f => f != null))
{
    tempFilesToCleanup.Add(adjustedFile);
}
```

#### AdjustItemsAudioSpeedAsync 方法（第923-961行）
```csharp
string adjustedFile = Path.Combine(_tempFolder, $"speed_item_{Guid.NewGuid()}.wav");
// ...
item.TempFile = adjustedFile;  // 这里修改了 item.TempFile
```

### 1.3 根本原因
当 `task.Speed == 0` 时，`AdjustItemsAudioSpeedAsync` 方法会修改 `item.TempFile` 为调整后的文件路径，但**原始的临时文件路径（TTL引擎生成的文件）丢失了**。

删除时只删除了：
1. `item.TempFile`（调整后的文件）
2. `tempFilesToCleanup`（合并文件等）

但**原始临时文件没有被删除**，因为它的路径已经被覆盖了。

---

## 二、修改方案

### 2.1 VoiceGenerationTaskQueue.cs

#### 2.1.1 在调整速度前保存原始临时文件路径
在 `ExecuteTaskAsync` 方法的 `task.Speed == 0` 分支中，**先将原始临时文件添加到清理列表**，然后再调整速度。

**修改位置**：第 559-578 行

**当前代码**：
```csharp
else
{
    task.ProgressDetail = "正在调整各段音频速度...";
    OnTaskProgressUpdated(task, task.Progress, task.ProgressDetail);

    var adjustedFiles = await AdjustItemsAudioSpeedAsync(task.Items);
    foreach (var adjustedFile in adjustedFiles.Where(f => f != null))
    {
        tempFilesToCleanup.Add(adjustedFile);
    }

    if (_shouldStop) return;

    task.ProgressDetail = "正在合并音频...";
    OnTaskProgressUpdated(task, task.Progress, task.ProgressDetail);

    mergedFile = await MergeAudioFilesAsync(task.Items, task.SpaceTime, task.SaveFile);
    tempFilesToCleanup.Add(mergedFile);

    speedAdjustedFile = mergedFile;
}
```

**修改为**：
```csharp
else
{
    // 先保存原始临时文件路径到清理列表（因为后续会被修改）
    foreach (var item in task.Items)
    {
        if (!string.IsNullOrWhiteSpace(item.TempFile) && File.Exists(item.TempFile))
        {
            tempFilesToCleanup.Add(item.TempFile);
        }
    }

    task.ProgressDetail = "正在调整各段音频速度...";
    OnTaskProgressUpdated(task, task.Progress, task.ProgressDetail);

    var adjustedFiles = await AdjustItemsAudioSpeedAsync(task.Items);
    foreach (var adjustedFile in adjustedFiles.Where(f => f != null))
    {
        tempFilesToCleanup.Add(adjustedFile);
    }

    if (_shouldStop) return;

    task.ProgressDetail = "正在合并音频...";
    OnTaskProgressUpdated(task, task.Progress, task.ProgressDetail);

    mergedFile = await MergeAudioFilesAsync(task.Items, task.SpaceTime, task.SaveFile);
    tempFilesToCleanup.Add(mergedFile);

    speedAdjustedFile = mergedFile;
}
```
